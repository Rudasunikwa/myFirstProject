1. Open folder where project is saved.
2. Go to location bar and type, cmd then press enter
3. when the terminal opens type the following commands
	git init
	git add .
	git commit -m "Initial Commit"
4. Go to git hub on the web browser, create a new repository
5. Copy the link to the repository
6. Continue typing the following git commands
	git remote add origin <git hub repository Link>
	git branch -M main
	git push -u origin main

Now your files should be uploaded unto your repository

7. Open github, click on settings
